# Unit Testing Practice Project

## English

This project is designed for practicing unit testing in the Postgraduate Course on Test Automation.

### Installation

Clone the repository and install dependencies:

```bash
npm install
```

### Usage

Run the tests:

```bash
npm test
```

### Project Structure

- `src/biblioteca.js`: Main library code
- `test/biblioteca.test.js`: Unit tests
- `mochawesome-report/`: Test reports

### Technologies

- Node.js
- Mocha
- Chai (assumed based on test structure)
